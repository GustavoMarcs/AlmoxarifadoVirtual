Almoxarifado Virtual é um projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) no SENAI, 
com o objetivo de digitalizar e otimizar o controle de materiais e estoque. O sistema foi projetado para ser genérico e reutilizável, 
podendo ser aplicado em qualquer tipo de empresa ou organização que necessite de gerenciamento de almoxarifado.

—

A aplicação foi desenvolvida inteiramente em Blazor Server, sem utilização de APIs externas, 
utilizando Entity Framework Core para acesso a dados e SQL Server como banco de dados. 
Essa abordagem garante simplicidade arquitetural, integração direta entre a interface e a camada de dados, 
além de bom desempenho em ambientes corporativos.

—

O objetivo do projeto é oferecer um sistema web funcional e organizado que permita o cadastro de produtos, 
a organização por categorias, o controle de entrada e saída de materiais, a consulta de estoque em tempo real 
e a redução de processos manuais e falhas operacionais.

—

As tecnologias utilizadas incluem Blazor Server, C# e .NET, Entity Framework Core, SQL Server, 
Razor Components e Domain-Driven Design (DDD), adotando boas práticas de organização e separação de responsabilidades.
